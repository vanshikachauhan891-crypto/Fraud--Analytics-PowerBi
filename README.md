# 💳 Financial Ecosystem & Fraud Analytics Dashboard

## 📌 Project Overview
This project features a high-impact **Power BI Dashboard** designed to monitor financial health and detect fraudulent activities. It analyzes over **$1.43 Billion** in transactions to provide actionable insights into regional performance, merchant categories, and security risks.

## 📊 Key Highlights & Insights
* **Transaction Summary:** Managed a total volume of **$1.43bn** with deep dives into revenue trends.
* **Fraud Monitoring:** Successfully identified **3,000 fraud cases**, maintaining a clear view of the **4.95% fraud rate**.
* **Global Reach:** Analyzed performance across key regions: **North America, Europe, and Africa**.
* **User Behavior:** Insights into transactions based on device types (Mobile vs. Desktop) and payment channels.

## 🛠️ Tools & Technologies
* **Power BI:** For advanced data visualization and interactive filtering.
* **Data Cleaning:** Processed and structured using **Excel/CSV**.
* **Analysis Techniques:** Trend analysis, KPI tracking, and Risk Assessment.

## 📂 Project Structure
* `Capstone project PowerBi.pbix`: The main interactive dashboard file.
* `Capstone Project Cleaned Data.xlsx`: The cleaned dataset used for analysis.
* `README.md`: Project documentation (this file).

## 🚀 How to View
1. Download the `.pbix` file from this repository.
2. Open it using **Power BI Desktop**.
3. Explore the interactive filters for different regions and merchant categories.
